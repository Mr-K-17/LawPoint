## Quick Start

### Prerequisites
- Node.js v14+
- MongoDB (local or cloud)
- npm or yarn

### Steps to Run

1. Extract the ZIP and navigate to project root
2. Run `npm run install-all` to install backend and frontend dependencies
3. Start MongoDB server (`mongod`)
4. Edit `backend/.env` to set up environment variables, including MongoDB URI
5. Run `npm run dev` to start backend and frontend servers concurrently
6. Open http://localhost:3000 to use the application
